## Facebook Groups Tools

### IMPORTANT: API Deprecation Notice
As of **April 2024**, Meta completely removed the Facebook Groups API. This means:
- No third-party apps can auto-post to groups
- All "scheduling" tools now use push notification workarounds
- Lead capture from groups you admin is still possible via browser extensions

### Compliance & Risk Warning

**Meta's Enforcement (2026):**
- Faster restrictions with fewer warnings
- AI detection flags: activity spikes, repeated message patterns, unusual logins
- Consequences: Warning → Feature restrictions → Account disabled

**Safe Practices:**
- Focus on groups you admin (lead capture is safe)
- Avoid aggressive automation tools
- Personalize outreach, don't mass-blast
