# AI Sales Coach - Complete Dashboard System Game Plan

## System Overview

A unified call center dashboard with real-time calling, call logging, missed call tracking, callback management, and full transcript history.

---

## Page Structure

### 1. Main Dashboard (`/dashboard`)
The central hub showing all call activity at a glance.

### 2. Live Calling Interface (`/call`)
Full-screen focused calling experience.

### 3. Call History (`/history`)
Searchable log of all calls with filters and transcripts.

### 4. Missed Calls & Callbacks (`/callbacks`)
Track missed calls and callback status.

### 5. Supervisor Dashboard (`/supervisor`)
Monitor live calls across all reps.
